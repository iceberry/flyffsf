Editing Bones

Load the desired OBJ file. Click Misc > Animated than Misc > Load Reference. Choose a O3D File that is similar to your object. The different colors represent different bones. You can now set all the bones that are used in the reference file. Caution: Some colors have a very slight difference.

You can use TAB to switch between your mesh and ref mesh to change the view with 1,2,3 keys.

Once the ref mesh is loaded you can use arrow keys left and right to cycle trough the points and up and down to change the bone for that point.

Example:

  1. You finish your mesh in eg. Cinema4D and export it to OBJ format.
  2. You load the OBJ into the converter.
  3. You click Misc. -> Animated
  4. You click Misc. -> Load reference
  5. You chose a existing mesh that looks close to the one you made yourself. So eg you made long boots than you need a refenece with long boots, too.
  6. Now you press left or right arrow key and also up or down arrow key. The points should now all turn into one color.
  7. Now you can chose points with left/right and change their color with up/down. You have the reference to the right so you see which region has which color. I again tell you check also the bonevalue in the upperright corner, because some colors look very similar. I will change that sometime ;)

Elemental glow

To enable elemental glow for you weapon, you have to add a new object group called "element" and give it 3 or 5 points. the 1st is the vertex (0,0,0) the following define start and end point of the first "element row" and the same for 2nd elemental row. The red lines are btw those "element rows". Export a O3D file to OBJ to see how it works..

Bugs

Some O3D files won't load at all.
Some O3D files consist of multiple meshes. You will only see the 1st one, so don't wonder if the mesh seems incomplete.
If your OBJ file is not in the correct format, loading it will fail.
